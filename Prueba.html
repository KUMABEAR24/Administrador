🚀 Configuración inicial (síguelo paso a paso)

  1. Ve a sheets.google.com → crea una hoja nueva → nómbrala MORITA SAS
  2. Copia el ID del URL (entre /d/ y /edit) y guárdalo
  3. Crea 3 pestañas con estos nombres exactos: Productos · Pedidos · Config
  4. Pestaña Productos fila 1: id · nombre · descripcion · precio · precio_anterior · imagen · categoria · stock · activo · badge
  5. Pestaña Pedidos fila 1: numero · fecha · nombre · apellido · email · telefono · ciudad · direccion · nota · productos_txt · subtotal · envio · total · metodo_pago · estado
  6. Pestaña Config fila 1: clave · valor → luego agrega: ultimo_pedido · 399 y envio_costo · 0
  7. En la hoja: Extensiones → Apps Script → pega el código del archivo apps-script.js → pon tu Sheet ID → Guardar
  8. Implementar → Nueva implementación → Aplicación web → Ejecutar como: Yo · Acceso: Cualquier persona → Copiar URL
  9. Pega esa URL en el campo de abajo Y también en index.html donde dice const GAS_URL = ''

🔗 URL de Apps Script
